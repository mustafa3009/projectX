Installation of docker & jenkins

1. Provision a centos 7 VM using Virtual Box. Use the minimal install ISO and launch screen 
2. sudo yum update -y
3. Docker installation - https://docs.docker.com/engine/installation/linux/docker-ce/centos/
	Also do the post installation steps https://docs.docker.com/engine/installation/linux/linux-postinstall/
4. sudo yum install git
5. Maven installation
6. Jenkins installation
7. Add jenkins to the docker group
	sudo usermod -a -G docker jenkins
